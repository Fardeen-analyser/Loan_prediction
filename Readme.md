<h1> <span style='color:purple'> 🏦 LOAN PREDICTION USING MACHINE LEARNING </h1>

### <span style='color:lightgreen'> 📘 Overview

 <span style='color:lightblue'> The Loan Prediction project aims to predict whether a loan applicant will be approved or not based on their personal and financial details. This project uses machine learning techniques to analyze applicant data and make predictions, helping financial institutions make faster and more accurate decisions.

 ### <span style='color:lightgreen'> 🎯 Objective

<span style='color:lightblue'> To build a machine learning model that predicts the likelihood of loan approval based on features such as:

<span style='color:lightblue'> Gender

<span style='color:lightblue'> Marital status

<span style='color:lightblue'> Credit history

<span style='color:lightblue'> Employment type

<span style='color:lightblue'> Dependent

<span style='color:lightblue'> Property Area

### <span style='color:lightgreen'> 🧩 Dataset

 <span style='color:lightblue'> Source: (e.g., Kaggle – Loan Prediction Dataset)

 <span style='color:lightblue'> Size: [Number of rows and columns, e.g., 367 rows × 9 columns]

 <span style='color:lightblue'> Target Variable: Credit_History (1 = Approved, 0 = Not Approved)

#### <span style='color:yellow'> Key Features:
</span>
<span style='color:lightblue'>
<li> Gender  -->  Male and Female
<li> Married -->  Married(Yes)/ Not Married(No)
<li> Dependents --> Number of persons depending on the client
<li> Self_Employed -->  Self Employed (Yes/No)
<li> LoanAmount -->  Loan Amount in thousands
<li> Loan_Amount_Term --> Term of of learn in months
<li> Credit_History --> Credit history meets guidelines
<li> Property_Area --> Urban, Semi and Rural
</span>
